Java中的常量

  • 常量的定义：在程序运行时保持不变的量就是常量
          比如在HelloWorld中的“Hello，World！！！！！”就是常量。
分类 定义 特点 例子
字符串常量 由“”（双引号）引起来的部分 一长串 “abc”“Hello”
字符常量 由‘’（单引号）引起来的一部分 只有一个 ‘a’‘1’‘中’
整型常量 直接写出的数字，没有小数点 。 5；100；-200
浮点型常量 直接写出的数字，有小数点 。 5.5;3.14
布尔常量 只有真假 。 true;false
空常量 没有任何数据，null 。 。

对第一个小程序的注释

  • Java 的注释和C语言差不多（没什么好注意的）

一些关键字

  • 关键字：有特殊含义的，不能随便使用
  • 关键字特点：
  • 1. 完全是小写字母
  • 2. 在一些编辑器中有特殊颜色（还是比较好认）

标识符

  • 标识符：在Java程序中，我们自己定义的内容，比如：类的名字，变量名等等。

    • HelloWorld小程序中的标识符就是第三个单词**HelloWorld**

  • 命名规则：（硬性要求）

    1. 可以包含英文字母（区分大小写）、0~9数字、_（下划线）、$（美元符号）。

    2. 不能以数字开头。
    3. 不能是关键字。

  • 命名规范：

    1. 类名称：首字母大写，每个单词的首字母大写。
    2. 变量名：首字母小写，后面每个单词首字母大写。
    3. 方法名：同上。

Java语言开发环境的搭建

  1. Java虚拟机—JVM
    • JVN是运行所有Java程序的假想计算机，是Java程序的运行环境，Java代码要运行在JVM虚拟机上。
    • 注：可以把JVM理解为各个系统和Java代码之间的翻译
  1. JRE和JDK
    • JRE：（Java Runtime Enviroment）：Java程序的运行环境，包含JVN和所需核心类库。
    • JDK：（Java Devolopment Kit）：是Java开发工具包，包含JRE和开发人员的使用工具。
