三元运算符

一元运算符：只有一个数据就可以运算，例如：++、–、！
二元运算符：两个数据才可以进行操作，例如：+、-、*、/、%、=
三元运算符：三个数据才能操作

格式：
数据类型 变量名 = 条件判断 ？ 表达式A ： 表达式B；

流程：
首先判断条件是否成立
如果为true，则把A赋给变量
如果为false，把B赋给变量

注意事项：
1. A、B必须符合左侧数据类型的要求
2. 三元运算的结果必须被使用：可以直接打印或交给一个变量

输出结果：

逻辑运算符

与 （并且）：&& –> 全都是true才是true,否则就是false
或 （或者）：|| –> 至少一个ture,就是true,全是false，才是false
非 （取反）：! –>本来是false变true，本来是true变false

与“&&”，或“||”，具有短路效果，如果左边已经得到结果，那么右边的代码就不执行了

注意事项

  1. 逻辑运算符只能用于boolean值
  2. 与、或左右都要有值，取反只要有一个值
  3. 与、或有多个条件时可以连着写
    例如：A && B && C

自增和自减

自增运算符：++
自减运算符：–

含义：让一个数字涨一，或降一
使用格式：可以写在变量名之前或之后l
例如：num++;++num
使用方式：
1. 单独使用：不和其他操作使用，自己独立为一步
2. 单独使用：和其它操作混合，例如：与赋值混合，与打印操作混合
两种方式的区别：
1.单独使用时：前++和后++没有区别
2. 混合使用时：有重大区别
A. 如果是++num，那么变量立刻+1，然后拿着结果使用
B. 如果是num++，那么使用变量本来的数值，然后+1
注意事项：
只有变量才能使用自增和自减，常量不能发生改变
